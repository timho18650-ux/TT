# The Westin Grande Sukhumvit 附近可購物商場整理

參考酒店座標：13.7388, 100.5601（Sukhumvit 區）

---

## 1. Terminal 21 Asok

| 項目 | 內容 |
|------|------|
| **距離** | 約 7 分鐘步行（約 395 呎） |
| **交通** | BTS Asok 站 1 號出口／MRT Sukhumvit 站 3 號出口，天橋直達 |
| **地址** | 88 Sukhumvit Soi 19（Sukhumvit 與 Asok 交叉口） |
| **營業時間** | 10:00–22:00 |
| **特色** | 航站主題，每層代表不同城市（東京、倫敦、舊金山、好萊塢等）；600+ 店鋪、Pier 21 美食廣場、SF 影城；平價年輕品牌與泰國設計師 |
| **介紹影片** | [Walking Terminal 21 Asok \| Bangkok's Airport-Themed Mall](https://www.youtube.com/watch?v=ZCI9JdmbBx8) |
| **另薦** | [4K Walk inside Terminal 21 Asok](https://www.youtube.com/watch?v=VmnbV_PJvMg) |

## 2. EmQuartier

| 項目 | 內容 |
|------|------|

## 3. Emporium

| 項目 | 內容 |
|------|------|
| **距離** | 約 0.82 英里／BTS 1 站 Phrom Phong |
| **交通** | BTS Phrom Phong 站，天橋直達 |

## 4. EmSphere

| 項目 | 內容 |
|------|------|
| **距離** | 約 12 分鐘步行／BTS Phrom Phong 站 |
| **交通** | BTS Phrom Phong 站，鄰近 Benchasiri Park（Sukhumvit 22–24 之間） |
